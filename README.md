## How to Use

### View Portfolio
Simply open `index.html` in your web browser.

### Generate Resume PDF
1. Open `resume.html` in your browser
2. Click the "Print / Save as PDF" button
3. Choose "Save as PDF" in the print dialog
4. Save as `resume.pdf` in the same folder

### Customize

#### Update Social Links
Edit `index.html` and replace:
- `YOUR_LINKEDIN` with your LinkedIn username
- `syedmasud@example.com` with your actual email
- Add your phone number in the resume

#### Add More Projects
In `index.html`, find the Projects section and add new project cards following the existing template.

#### Modify Colors
Edit `style.css` and change the CSS variables at the top:
```css
:root {
    --primary: #6366f1;
    --secondary: #8b5cf6;
    --accent: #ec4899;
}
```
